Chapter 2 Summary - Programming Basics

  • There are six primitive data types: strings, symbols, numbers, Booleans, undefined and null.
  • Non-primitive data types, such as arrays, functions and objects, all have a type of 'object'.
  • Use the "typeof" operator to find out the datatype of a value.
  • Variables point to values stored in memory and are declared using the const or let keywords.
    ** We dont use var anymore.
    Use const when the variable will not be reassigned to another value. (Constant)
    Use let if the variable might be reassigned later in the program. (Can change - the new 'var')
  • Backkslashes are the escape mark - used to escape special characters in strings:
    -single quote \', -double quote \", -end of line \n, -carriage return \r, -tab \t, backslash \\
  • Template literals: use BACKTICKS (`) instead of quote marks for string literals.
    This allows JS expressions to be inserted inside a string.
    example: const age = 39;
    `I will be ${age + 1} next year!`
    output: "I will be 40 next year!"
  • Symbols are unique, immutable values.
    The only way to create them is to use the symbol() function.
  • There are only seven values that are false in JavaScript and these are known as 'falsy' values.
  • Data types can be converted into other data types.
  • Type coercion is when JavaScript tries to convert a value into another data type in order to perform an operation.
  • Logical operators can be used to check if compound statements are true or false.
  • Values can be compared to see if they are equal, greater than or less than other values.

Chapter 3 Summary - Arrays, Logic, and Loops

  • Arrays dont have to use the same data types inside of it.
  • Destructuring Arrays: the concept of taking values out of an array and
    presenting them as individual values.
  • Declaring an array:
    const avengers = ['Capt America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow'];

Properties & Methods

  • To find the length: averngers.length;
  • To remove the last item of the array: avengers.pop();
  • To remove the first item of the array: avengers.shift();
  • To append a new value to the end: avengers.push();
  • To append a new value to the beginning: avengers.unshift();
  • You can use the concat() method to merge with one or more arrays.
    *Note that this does not change the array, it creates another array.
    You can use the assignement (=) to update the array to this new array:
    ex: avengers = avengers.concat(['Batman', 'Spiderman', 'Green Lantern']);
  • The join() method can be used to turn the array into a string that
    comprises all the items in the array, separated by commas.
    ex: avengers.join();
    You can also choose a different seperator like '&' by placing it in the ().
    ex: avengers.join('&');
  • The slice() method creates a subarray. It is non-destructive, so no
    items are removed from the array, it just shows you what thse items are.
    ex: avengers.slice(2,4) //this starts at the 3rd item (index 2)
    // and it finishes at index 4 (not inclusive). Output: "['Thor', 'Hulk']"
  • The splice() method REMOVES the items, then inserts new items in their place.
    ex: avengers.splice(3, 1, 'Scarlet Witch');
    *This method is destructive as it changes the values of the array.
    This first number in this method tells us the index at which to start
    the splice (index 3). The second number tells us how many items to remove (1).
    And then the 3rd thing is the insert.
  • You may also use splice() to insert items at a specific index without
    removing anything by indicating 0 items to be removed in the second
    number position.
    ex: avengers.splice(4, 0, 'Quicksilver');
  • You may also use splice() to delete items without inserting new.
    ex: avengers.splice(2, 1);
    *This will remove 1 item starting at index 2 completely.

Other stuff:

  • Multidimensional arrays:
    ex: const coordinates = [[1,3],[4,2]];
  • To access the values in Multidimensional arrays, we use two indices:
    One to refer to the items place in the outer array, and one to refer
    to its place in the inner array.
    examples:
    coordinates[0][0]; //1st value of the 1st array
    coordinates[1][0]; //1st value of the 2nd array
    coordinates[0][1]; //2nd value of the 1st array
    coordinates[1][1]; //2nd value of the 2nd array
  • Sets: A set is a data structure that represents a collection of unique
    values, so it cannot include any duplicate values.
  • To create one: const list = new Set();
    To add values: list.add(1);
    Add multiple items: list.add(2).add(3).add(4);
    -OR-
    const numbers = new Set([1,2,3]);
  • Anything repeated will only appear once in the set. This give a
    convenient way of eliminating duplicate values from an array with
    a single operation.
  • If a string is used as teh argument in a set, each character will
    be added separately with repeated chars ignored.
  • To add separate words to a set, you need to use the add() method.
    ex: const words - new Set().add('the').add('quick').add('fox');
  • You can do this with arrays too:
    const arrays = new Set().add([1]).add([1]);
    *The arrays may look the same, but are considered different objects.
  • Set Methods:
    -To find the # of values in a set = size(); ex: numbers.size();
    -To check if a value is present = has(); ex: numbers.has(7);
    -To remove values from a set = delete(); ex: numbers.delete(2);
    -To remove ALL values from a set = clear(); ex: numbers.clear();
  • To convert a set into an array:
    -Use the spread operator (...) directly inside the array literal.
    ex: const shoppingArray = [...shoppingSet]
  • Maps: a convenient way of keeping a list of key and value pairs.
    Similar to 'hashes', 'hash tables', or 'dictionaries' in other pl's.

Looping over arrays:

  • for(let i = 0, max = avengers.length; i < max; i++){
    console.log(avengers[i]); }
    *Setting a max variable is preferable to using "i < avengers.length"
    because then the length of the array would have to be calculated
    after every pass thru. This makes a difference to speed
    when using large arrays.
  • New iterator function for arrays: THE for-of LOOP:

    for(const value of avengers){
    console.log(value); }
  • This replaces all the setup of a for loop with a variable that
    represents the value of each element in the array. Note that
    this variable needs to be declared using "const".

Chapter 4 Summary - Functions

  • Function Declarations:
    ex: function hello(){
    console.log('Hello World!'); }
  • Function Expressions:
    (this one assigns an anonymous function to a variable)
    ex: const goodbye = function(){
    console.log('Goodbye World!'); }
    (this one aassigns a named function to a variable
    ex: const goodbye = function bye(){
    console.log('Bye-bye!'); }
  • The difference between parameters and arguments:
    -Parameters a function needs are set when the function
    is DEFINED.
    -Arguments are passed to the function when it is INVOKED.
  • To have a variable number of arguments: use the rest operator (...)
    This will collect all teh arguments entered into an array.
    ex: function mean(...values){
    let total = 0;
    for(const value of values){
    total += value; }
    return total/values.length; }
  • Default parameters: a value set to be used if no arguments are provided.
    ex: function hello(name = 'World'){
    console.log(`Hello ${name}!`); }
    -If no argument is passed: hello();
    output = 'Hello World!'
    -If argument IS passed: hello('Cheryl'); output = 'Hello Cheryl!'
  • Default parameters should ALWAYS come after non-default parameters:
    ex: function discount(price, amount = 10){
    return price * (100-amount)/100; }
    - discount(20); //standard discount of 10% on $20
    - discount(15, 20); //discount of 20% on $15
  • Arrow Functions: uses less syntax for declaring functions.
    - Identified with the arrow symbol (=>).
    - The parameters come before the =>, and the main body
    of the function comes after.
    - They are always anonymous, so you have to assign them to a variable.
    ex: const square = x => x * x;
    const hello = () => alert('Hello!);
    const add = (x, y) => x + y;
    - These dont need block {} or return keywords.
    - If the function ismore than one line, this method is not practical.
    Best for short, anonymous functions.
  • Callbacks: A function that is passed as an argument to another function.
    The callback is provided as a parameter and invoked inside the body.
    ex: function sing(song, callback){
    console.log(`I'm singing along to ${song}`);
    callback(); }

    function dance(){
    console.log("I'm moving my body to the groove."); }

    -Now we call our sing function and callback dance:
    sing('Let It Go', dance);
    - output: 'I'm singing along to Let It Go'
    'I'm moving my body to the groove.'
